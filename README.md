# VN-localisation
Surname localisation:  
  
Turns the following  
1 = {  
    prefix = "dynnp_von"  
    name = "dynn_Eppenstein"  
    culture = austrian  
}  
into the  
dynn_Eppenstein: "Eppenstein"  
  
Title localisation:  
  
Extracts everything starting with "e_, k_, d_, c_, b_" and turns it into e_hre: "Hre"
